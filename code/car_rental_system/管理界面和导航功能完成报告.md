## 技术实现

### 框架和库
- **前端框架**: Bootstrap 5.3.0
- **图标库**: Font Awesome 6.4.0
- **JavaScript**: 原生JavaScript + Bootstrap JS
- **后端**: Django 5.2.8

### 响应式设计
- 桌面端：侧边栏固定 + 主内容区域
- 平板端：自动适应
- 移动端：折叠导航 + 全屏内容

### 状态管理
- 侧边栏折叠状态本地存储
- 统一的错误处理机制
- Ajax请求状态管理

### 数据展示
- 卡片布局统计信息
- 颜色编码状态指示
- 分页和搜索支持
- 实时数据更新

## 访问地址
- 主页（仪表板）：http://localhost:8000/
- 车辆管理：http://localhost:8000/vehicles/
- 客户管理：http://localhost:8000/customers/
- 租赁管理：http://localhost:8000/rentals/
- 管理后台：http://localhost:8000/admin/
